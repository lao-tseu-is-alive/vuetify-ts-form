# vuetify-ts-form

Generate a dynamic form based on a basic json schema.


## Project setup

```
# npm
npm install
```

### Compiles and hot-reloads for development

```
# npm
npm run dev
```

### Compiles and minifies for production

```
# npm
npm run build
```
